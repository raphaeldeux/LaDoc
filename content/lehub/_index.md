# LeHub

LeHub est l'intranet de La Fresque Systémique, accessible sur `hub.fresquesystemique.org`. C'est aussi le moteur de tout le SI : sa base de données fait référence pour les membres, les ateliers, les inscriptions, les contenus éditoriaux et la configuration de l'association.

## À qui il s'adresse

| Profil | Ce qu'il y fait |
|---|---|
| **Animateur·ice** | Créer et gérer ses ateliers, pointer les présences, accéder aux supports pédagogiques |
| **Formateur·ice** | Tout ce que fait un animateur, plus les supports de formation |
| **Admin** | Gérer les membres, les événements, les contenus, les paramètres et les intégrations |

L'accès demande un compte membre. Les adhérents simples n'ont pas vocation à s'y connecter : le niveau d'accès minimum visé est animateur·ice.
